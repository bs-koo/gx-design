# Design Strategy Skill

## 목적

리서치 자료를 실제 디자인과 콘텐츠 제작을 지휘할 수 있는 명확한 전략으로 변환한다.

## 수행 절차

1. 프로젝트 목표와 비즈니스 목표를 확인한다.
2. 사용자 문제와 행동 맥락을 정의한다.
3. 리서치에서 가장 중요한 인사이트를 선택한다.
4. 문제, 원인, 기회를 구분한다.
5. 가치 제안과 차별화 방향을 수립한다.
6. 디자인 원칙을 3~5개로 정의한다.
7. 크리에이티브 콘셉트를 개발한다.
8. 매체별 적용 방식을 구체화한다.
9. 결과물과 제작 우선순위를 정의한다.
10. 성공 기준과 리스크를 정리한다.

## 좋은 전략의 조건

- 목표와 직접 연결된다.
- 특정 사용자와 상황을 전제로 한다.
- 경쟁 대안과 차이가 있다.
- 디자인 선택의 기준이 된다.
- 제작자가 해석하지 않고 실행할 수 있다.
- 무엇을 하지 않을지도 설명한다.
- 결과를 평가할 기준이 있다.

## 콘셉트 작성 형식

- Concept Name
- One-line Definition
- Core Insight
- Strategic Rationale
- User Value
- Visual Expression
- Verbal Expression
- UX Expression
- Motion Expression
- Risks
