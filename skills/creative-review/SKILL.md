# Creative Review Skill

## 목적

완성된 결과물이 단순히 보기 좋은 수준을 넘어 프로젝트 목표와 전략을 충족하는지 검수한다.

## 공통 검수 기준

### 목표 적합성

- 프로젝트 목표를 충족하는가?
- 타깃 사용자에게 적절한가?
- 핵심 메시지가 명확한가?

### 전략 일치성

- 승인된 전략과 일치하는가?
- 핵심 인사이트가 결과물에 반영되었는가?
- 차별화 방향이 드러나는가?

### 정보 구조

- 정보 우선순위가 명확한가?
- 불필요한 요소가 있는가?
- 사용자가 다음 행동을 이해할 수 있는가?

### 시각 완성도

- 위계가 명확한가?
- 레이아웃과 간격이 일관적인가?
- 컬러와 타이포그래피가 체계적인가?
- 이미지 스타일이 일관적인가?

### 생성·렌더 실물 (실물 이미지가 있는 경우)

- 실물 이미지를 직접 열어 확인했는가?
- AI 아티팩트나 렌더 결함이 없는가? (깨진 글자, 왜곡, 폰트 폴백 흔적)
- 기술 규격이 정확한가? (치수, 해상도, 컬러 값)
- 구성 문제가 없는가? (잘림, 오버랩, 여백 불균형)
- 다른 자산과 스타일이 일관적인가?
- 산출물이 의도와 다른 것으로 읽히지 않는가? (정체성 오독)

### UX/UI

- 핵심 과업이 효율적인가?
- 상태와 피드백이 명확한가?
- 오류와 예외 상황을 고려했는가?
- 접근성과 반응형을 고려했는가?

### 영상

- 초반에 관심을 확보하는가?
- 메시지 흐름이 자연스러운가?
- 장면과 내레이션이 중복되지 않는가?
- 자막의 가독성과 노출 시간이 적절한가?
- 러닝타임에 불필요한 장면이 없는가?

### 제작 가능성

- 실제 제작 가능한 수준인가?
- 필수 규격이 포함되어 있는가?
- 파일 및 결과물 구조가 명확한가?
- 추가 결정이 필요한 부분이 표시되어 있는가?

## 검수 결과 형식

1. 총평
2. 잘된 점
3. Critical Issues
4. Major Issues
5. Minor Issues
6. 수정 권장사항
7. 수정 우선순위
8. 최종 승인 여부
